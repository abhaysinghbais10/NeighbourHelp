# Phool Basket App
React Native (Expo) mobile application for a premium gifting store.

## 📁 Project Structure

- **`/assets`**: Contains images and branding assets (logo, icons, etc.)
- **`/components`**: Reusable UI parts (Navbar, Product Cards, Sliders)
- **`/context`**: Global state management (Cart, Favorites, User, Payments, Addresses)
- **`/data`**: Mock data for products and categories
- **`/screens`**: All main app pages (Home, Checkout, Profile, Orders, etc.)
- **`App.js`**: Main entry point where navigation and global providers are configured

## 🚀 Key Features
- **Branding**: Official Phool Basket logo and styling integrated.
- **Shopping**: Browse products, search, and add to cart/favorites.
- **Checkout**: Multi-step checkout with address and payment management.
- **Account**: User profile, order history, and saved payment methods.

## 🛠️ Editing Tips
- **To change the logo**: Update `assets/logo.jpg`.
- **To add a screen**: Create it in `screens/` and register it in `App.js`.
- **To edit the header**: Modify `components/Navbar.js`.
- **To update products**: Edit `data/products.js`.
